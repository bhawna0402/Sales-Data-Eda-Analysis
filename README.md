Sales Data Exploratory Data Analysis (EDA)
📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on a sales dataset to understand revenue trends, profitability drivers, and unusual transaction patterns.
The analysis focuses on identifying business insights through data cleaning, visualization, and statistical exploration.
🎯 Objectives
Prepare and clean the dataset
Analyze sales trends over time
Examine relationships between Sales, Profit, and Quantity
Identify outliers in profit
Derive actionable business insights
📁 Dataset Description
The dataset contains transactional sales data including:
Order Date
Sales Amount
Profit
Quantity
Category
Customer and location details
The data is stored in CSV format and analyzed using Python.
🛠 Tools & Technologies Used
Python
Pandas – Data cleaning and manipulation
NumPy – Numerical computations
Matplotlib – Data visualization
Seaborn – Statistical visualization
📊 Analysis Performed
1. Data Preparation
Checked for missing values
Removed duplicate records
Converted date column to datetime format
2. Descriptive Statistics
Analyzed distribution of Sales, Profit, and Quantity
3. Category-wise Sales Analysis
Identified top-performing product categories
4. Sales Trend Analysis
Examined monthly sales trends
Observed seasonal patterns
5. Correlation Analysis
Studied relationships between Sales, Profit, and Quantity
6. Outlier Detection
Identified extreme profit values using IQR method
🔍 Key Insights
Revenue growth supports profitability, but margin management is essential.
Increasing sales volume alone does not significantly improve profit.
High-profit outliers indicate potential strategic opportunities.
📌 Conclusion
The analysis provides meaningful insights into sales performance, profit behavior, and transaction patterns. These findings can support forecasting, pricing optimization, and strategic decision-making.
