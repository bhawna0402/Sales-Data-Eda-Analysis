📊 Sales Data – Exploratory Data Analysis (EDA)
Overview
This project performs Exploratory Data Analysis (EDA) on a Sales Dataset to identify trends, relationships, and unusual transaction patterns. The objective is to derive actionable business insights that support data-driven decision-making.
Objectives
Clean and prepare the dataset
Analyze sales trends over time
Examine relationships between key variables
Identify outliers in profit
Derive strategic business insights
Dataset Description
The dataset contains sales transaction records including:
Order Date
Sales Amount
Profit
Quantity
Category
Customer and location details
The data is stored in CSV format and analyzed using Python.
Tools & Libraries Used
Python
Pandas – Data manipulation
NumPy – Numerical operations
Matplotlib – Data visualization
Seaborn – Advanced statistical visualization
Analysis Performed
1. Data Preparation
Checked for missing values
Removed duplicates
Converted date columns to datetime format
2. Descriptive Statistics
Examined mean, distribution, and variability of Sales, Profit, and Quantity
3. Category-wise Sales Analysis
Identified top-performing product categories
4. Sales Trend Analysis
Analyzed monthly sales patterns
Identified seasonal fluctuations
5. Correlation Analysis
Studied relationships between Sales, Profit, and Quantity
6. Outlier Detection
Identified extreme profit transactions using IQR method
Key Insights
Revenue growth contributes to profitability, but maintaining margins is essential.
Increasing sales volume alone does not significantly improve profit.
High-profit outliers indicate potential premium or strategic opportunities.
Conclusion
The analysis highlights important sales patterns, profitability drivers, and unusual transaction behavior. These insights can support forecasting, pricing strategy, and business growth planning.
